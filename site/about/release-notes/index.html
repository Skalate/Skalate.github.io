升级

要将MkDocs升级到最新版本，请使用pip：

pip install -U mkdocs

您可以使用mkdocs --version确定当前安装的版本：

$ mkdocs --version
mkdocs, version 1.0 from /path/to/mkdocs (Python 3.6)
