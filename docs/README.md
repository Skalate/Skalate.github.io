## My Wiki


一直以来，我用Microsoft Word做笔记，使用MathType插件撰写公式，这不是纯文本写的，要用鼠标点击各个符号，学习成本低，但是不支持大量公式呀，而且生成的公式是png格式的图片，这样排版起来很麻烦。后来使用Office自带的公式系统，最初它很不好用，最近2年（2020-2021）优化了不少。

后来使用LaTeX写笔记，安装了TeXLive 2020（5GB），使用VSCode作为编辑器，但编译时会生成很多无用的文件，整理起来比较麻烦。后来改安装了MikTek，安装包小了很多，额外的包会在调用时下载，且VSCode配置文件是一模一样的。

后来又用Pandoc，使用Markdown语法进行书写文档，然后run一下生成PDF格式的文件（此步需要调用LaTeX功能），由于用了模板，其实生成的PDF文件也挺美观的。最终的解决方案就是Markdown+VSCode+Pandoc+LaTeX，但是对于成系统的笔记，之前都是用Onedrive以文件的形式呈现，跨平台体验感不太好。

于是我开始探究如何使用静态网页搭建自己的Wiki，最初使用的是MkDocs，基于Python的一个开源库，会自动根据markdown文件生成静态网页文件。后来又转向了Docsify，它不会生成很多文件，但是功能就少了很多，而且不支持数学公式显示，这就很另人头大。
